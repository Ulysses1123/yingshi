# 萤石视频监控与配送事件系统 — 用户使用手册

> 本文档面向普通用户，不涉及任何技术细节。

---

## 一、系统简介

萤石视频监控与配送事件系统是一套为门店管理者设计的综合工具，提供两大核心功能：

1. **视频监控** — 实时查看各门店的摄像头画面，支持单屏/四屏切换和云台回放
2. **配送事件监控** — 查看各门店的订单配送状态（骑手到店、配送中等），并可直接跳转回放对应时间段的视频

---

## 二、如何访问系统

打开电脑上的浏览器（推荐 Chrome 或 Edge），在地址栏输入系统地址。

*示例地址格式：`http://服务器IP:5173`*

如果您不清楚访问地址，请联系 IT 管理员获取。

│              │                               │              │
│   门店列表    │        视频监控区域             │  配送事件面板  │

| 区域 | 位置 | 功能 |

## 四、门店树操作

左侧的门店列表以"城市 → 门店"的层级结构展示：

1. **展开/收起城市** — 点击城市名称（如"上海"）可展开或收起该城市下的门店列表
2. **选择门店** — 点击任意门店名称，系统会自动加载该门店的摄像头画面和配送事件
3. **当前门店标识** — 选中的门店会高亮显示

> 切换门店后，视频区域会自动切换到新门店的摄像头，配送面板也会更新为该门店的数据。

## 五、视频监控

### 5.1 观看实时画面

选定门店后，中央区域会自动显示该门店的摄像头画面。每个摄像头画面顶部会显示：
- 摄像头名称（如"A区"、"门口"、"打包台"等）
- 连接状态（绿色"● 直播中"或灰色"○ 未连接"）

### 5.2 切换单屏 / 4分屏

工具栏中有一个"单屏"和"4分屏"切换按钮：

- **单屏模式** — 放大显示第一个摄像头，适合仔细查看某个画面
- **4分屏模式** — 同时显示最多4个摄像头画面，适合全局监控

切换方式：直接点击"单屏"或"4分屏"按钮即可。

在4分屏模式下，**双击**任意摄像头画面可将其放大到单屏模式。

### 5.3 数码变焦

在摄像头画面上有一个"放大镜"图标按钮，点击后可启用**数码变焦**功能：

1. 点击放大镜按钮进入变焦模式
2. 画面会放大，并显示当前放大倍数
3. 再次点击放大镜按钮退出变焦模式

> 数码变焦是在画面上直接放大，不是摄像头的物理变焦。

## 六、配送事件面板

配送事件面板是系统的核心功能之一，用于查看门店的订单配送状态。

### 6.1 打开配送面板

选择门店后，配送面板会自动在右侧打开。您也可以通过工具栏中的**"配送"按钮**手动打开或关闭面板。

### 6.2 选择查询日期

配送面板上方有一个**日期选择框**，默认显示当天日期。

- 点击日期框可以弹出日历，选择任意日期
- 选择日期后，系统会自动查询该门店当天的配送订单数据
- 配送数据来源于接力平台（与门店对接的外卖配送平台）

> 可以查看**任意历史日期**的配送记录，不仅限于当天。

### 6.3 查看配送订单

配送面板会以卡片形式列出每个配送订单，每个订单卡片包含：

| 信息 | 说明 |
|------|------|
| **订单编号** | 该订单的平台订单号 |
| **状态标签** | 显示该订单当前状态（骑手到店 / 配送中 / 已送达 / 已取消 等） |
| **配送方式** | 如"京东秒送"、"蜂鸟众包"、"顺丰"等 |
| **时间线** | 按时间顺序显示该订单的配送事件 |

### 6.4 理解订单状态

系统使用不同颜色标识订单状态：

| 状态 | 颜色 | 含义 |
|------|------|------|
| **骑手到店** | 绿色 | 骑手已到达门店，等待取货 |
| **配送中** | 蓝色 | 骑手已取货出发，正在配送途中 |
| **已送达** | 绿色 | 订单已成功送达顾客 |
| **待接单** | 橙色 | 订单已创建，等待骑手接单 |
| **已取消** | 红色 | 订单已被取消 |

### 6.5 配送时间线说明

每个订单卡片内部有一条垂直的时间线，展示该订单的关键事件节点：

- **已下单** — 顾客已提交订单
- **🔵 骑手到店** — 骑手到达门店（绿色圆点标记）
- **🔵 配送中** — 骑手取货出发（蓝色圆点标记）
- **已完成** — 订单完成配送

> **点击时间线上的"已到店"或"配送中"事件**，系统会自动将所有摄像头切换到**回放模式**并跳转到该事件发生的精确时刻。

### 6.6 回放配送过程

如果订单既有"骑手到店"事件也有"配送中"事件，订单卡片底部会显示一个**"▶ 回放配送过程"按钮**。

点击该按钮，系统会自动：
1. 切换到回放模式
2. 设置开始时间为骑手到店时刻
3. 设置结束时间为骑手出发配送时刻
4. 播放该时间段内的视频画面

> 此功能需要门店安装了"门口"摄像头才能看到有价值的画面。

## 七、视频回放功能

### 7.1 手动进入回放模式

点击工具栏中的**"回放"按钮**进入回放模式。进入回放模式后，会显示额外的控制选项：

- **日期选择** — 选择要回放哪一天的视频
- **小时选择** — 选择从哪个小时开始回放（00:00 ~ 23:00）

### 7.2 回放模式下的操作

在回放模式下，视频画面会播放所选时间段的历史录像。您可以：

- 通过配送面板点击「已到店」或「配送中」事件，自动跳转到精确秒级的时刻
- 自行选择日期和时间来浏览历史画面

### 7.3 退出回放模式

再次点击"回放"按钮（或切换门店），系统会自动退出回放模式，回到实时画面。

### 7.4 存储类型说明

部分门店同时支持**云存储**和**本地存储**两种录像方式：

- **云存储** — 录像保存在萤石云服务器上，可远程访问
- **本地存储** — 录像保存在门店的存储设备（如SD卡、NVR）中

点击工具栏中的存储切换按钮可在两者之间切换。

> 如果不确定您的门店使用哪种存储方式，请联系管理员。

## 八、常用操作流程

### 场景一：查看某门店今天有哪些订单正在配送

1. 在左侧门店列表中点击该门店
2. 右侧配送面板自动打开，显示今天的数据
3. 查看配送订单列表，注意标有"骑手到店"或"配送中"的订单
4. 点击时间线上的「已到店」或「配送中」事件可跳转查看该时刻的监控画面

### 场景二：回放昨天某个订单的配送过程

1. 在左侧门店列表中选择门店
2. 在右侧配送面板的日期选择框中，选择昨天的日期
3. 系统加载该门店昨天的配送数据
4. 找到目标配送订单
5. 点击订单底部的"▶ 回放配送过程"按钮
6. 系统自动回放骑手到店至配送出发期间的视频

### 场景三：查找某个时间段的异常情况

1. 选择门店
2. 点击"回放"按钮进入回放模式
3. 选择日期和开始小时
4. 通过配送面板的「已到店」或「配送中」标记定位到可疑时刻
5. 或手动拖动视频进度查找

## 十、常见问题

**Q: 页面打开后没有画面显示？**

A: 请先在左侧门店列表中点击一个门店。如果没有门店列表或列表为空，请联系管理员检查系统配置。

**Q: 配送面板显示"暂无配送订单"？**

A：可能原因：
- 所选门店当天没有配送订单
- 该门店未接入配送平台
- 日期选择有误，可尝试切换日期

**Q: 视频画面一直显示"直播中"但没有图像？**

A: 可能是网络问题或摄像头离线，可尝试：
- 检查摄像头连接状态
- 刷新页面重试
- 如问题持续，请联系管理员检查摄像头设备

**Q: 点击配送事件后视频没有跳转？**

A：跳转需要在回放模式下工作。如果当前不在回放模式，点击事件会自动切换到回放模式。请确保：
- 摄像头支持回放功能
- 选定的日期范围内有录像文件

**Q: 4分屏模式下画面太小看不清？**

A: 可以**双击**任意画面将其放大到单屏模式，或者使用画面上的数码变焦功能。

**Q: 如何查看其他日期的配送记录？**

A: 在配送面板上方的日期选择框中点击，选择您想查看的日期即可。
