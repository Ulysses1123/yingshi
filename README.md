# 萤石视频监控播放器 — 用户说明

## 系统简介

多门店视频监控统一查看平台，基于萤石云(EZVIZ) SDK，支持实时直播、录像回放、电子放大等功能。适用于连锁门店的集中视频管理。

## 启动与访问

### 开发环境启动

后端和前端需要同时运行：

```bash
# 终端 1 — 启动后端 API 服务（端口 3000）
cd backend
npm install      # 首次需要
npm run dev

# 终端 2 — 启动前端界面（端口 5173）
cd yingshi
npm install      # 首次需要
npm run dev
```

打开浏览器访问 **http://localhost:5173**

> 后端自带内存降级模式，即使没有 MySQL 和 Redis 也能正常启动运行（使用内置的示例数据）。

### 准备工作

1. 在 `backend/.env` 中配置萤石云 AppKey 和 AppSecret：
  
  ```
  YS_APP_KEY=你的AppKey
  YS_APP_SECRET=你的AppSecret
  ```
  
2. 摄像头的验证码（validateCode）已配置在后端数据中，播放时自动注入

## 主要功能

### 📺 观看直播

1. 点击左侧任意门店
2. 右侧自动加载该门店的摄像头，以 4 分屏显示
3. 视频自动开始播放

### 🔍 单屏 / 4分屏切换

- 点击工具栏的 **单屏** 按钮 → 切换到单路画面，使用高清流
- 点击 **4分屏** 按钮 → 回到多路画面，使用流畅流
- **双击**4分屏中的任意画面 → 放大该路到单屏

### 🔎 电子放大

每个播放器右下角有放大镜按钮 `🔍+`：

- 点击启用 → 按钮变蓝，显示当前倍率（如 `2.5x`）
- 鼠标在视频上拖动选择放大区域
- 再次点击关闭放大
- 支持 SDK 工具条的滚轮缩放

### 📼 录像回放

1. 选择门店后，点击工具栏的 **回放** 按钮（橙色高亮表示激活）
2. 展开时间选择器：
  - **日期**：选择回放日期
  - **小时**：选择开始小时（从该小时 00:00 开始播放到 23:59）
3. 视频自动切换到回放模式
4. 使用 SDK 工具条控制播放/暂停/快进
5. 再次点击 **回放** 按钮恢复到实时直播

> ⚠️ 回放需要摄像头支持云存储或本地录像，如无录像内容会显示黑屏或错误。

## 常见问题

### Q: 视频一直显示"正在连接视频流..."

- 检查后端是否正常运行（浏览器访问 http://localhost:3000/health）
- 检查萤石云 AppKey/AppSecret 配置
- 摄像头是否在线

### Q: 提示"视频流连接失败"

- 点击 **重试** 按钮重新连接
- 验证码是否正确（已配置在后端数据中）
- 摄像头网络是否正常

### Q: 提示"同时预览路数达到上限"

- 萤石云免费账号通常限制 3-4 路同时预览
- 需要升级萤石云套餐或等待其他通道断开

### Q: 回放没有画面

- 确认该摄像头已开启云存储或录像功能
- 检查选择的日期是否有录像数据
- 尝试选择更早的日期
